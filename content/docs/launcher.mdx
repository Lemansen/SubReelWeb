# Лаунчер Subreel

Лаунчер нужен для быстрой установки клиента и удобного входа в экосистему проекта.

## Что умеет

- Быстрый запуск подготовленной сборки
- Управление модами и настройками
- Доступ к новостям проекта
- Переход к серверу без ручного ввода IP

## Установка

1. Открой страницу [Лаунчер](/launcher).
2. Скачай версию для Windows.
3. Запусти `SubReel.exe`.
4. Разреши приложению установить нужные файлы.

## Если у тебя не Windows

Сейчас основная версия ориентирована на Windows. За новостями по другим платформам следи в Discord и в релизах GitHub.

## Поддержка

Если лаунчер не запускается или не обновляется, напиши в Discord проекта и приложи описание ошибки.
